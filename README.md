# TODO Application
Описание

Консольное TODO-приложение для управления списком задач.
Проект позволяет закрепить навыки ООП, Collections, Streams, обработку ошибок и юнит-тесты.

Реализована слоистая архитектура: Controller → Service → Repository.


Технологии
Java 17+
Collections + Streams API
Lombok
JUnit
Maven
Git

Функционал

Просмотр всех задач
Добавление, редактирование и удаление задач
Изменение статуса (CREATED / IN_PROGRESS / DONE)
Фильтрация и сортировка задач
Команды: add, list, edit, delete, filter, sort, exit
